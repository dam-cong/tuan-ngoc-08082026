# Thiệp Cưới Online - Trần Tuân & Khánh Ngọc

Website thiệp cưới online cho đám cưới của **Trần Tuân** và **Khánh Ngọc** vào ngày **08/12/2026**.

## Tính năng

- Trang mở thiệp với hiệu ứng hoạt hình
- Đếm ngược tới ngày cưới
- Thông tin lịch trình sự kiện
- Album ảnh cưới
- Form RSVP (xác nhận tham dự)
- Sổ lưu bút (lời chúc)
- Thông tin quà mừng cưới (modal tài khoản ngân hàng)
- Nhạc nền tự động phát

## Công nghệ

- HTML/CSS/JS tĩnh
- Google Fonts
- Font chữ tùy chỉnh (UVN Hoa Tay, High Spirited, Lora, ...)

## Cấu trúc thư mục

```
assets/     # Fonts và hình ảnh
audio/      # Nhạc nền
elements/   # Hoa văn, icon, ảnh nền, phong bì
uploads/    # Ảnh cưới
```
